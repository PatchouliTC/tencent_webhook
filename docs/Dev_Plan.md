# TencentGit_WebHookRobotService

## 目标
 实现一个面向腾讯工蜂的webhook机器人服务
 
 该服务提供多存储库监听，基本信息本地快照存储，定时数据统计，多接收端发送等功能

 当前预计提供push,merge信息监听
